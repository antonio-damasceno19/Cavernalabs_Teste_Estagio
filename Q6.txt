React
É uma biblioteca JavaScript open source (código aberto) para criar interfaces de páginas web. Alguns sites que utilizam o React, como Netflix, Walmart;
Recentemente em 2015 o Facebook anunciou um módulo chamado React Native, que em conjunto com o React, possibilita o desenvolvimento de aplicativos para Android e iOS.
